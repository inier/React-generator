# React-generator

## Intro
React-generator client or ssr for development

* webpack 4.5
* koa 2
* Redux
* react-router 4
* sass
* px compile for vw
* 字符串转码国际化解决方案（编译后转码）
* 基础代码规范遵照的 Airbnb

## Usage
```
# clone
git clone https://github.com/yuanzhhh/React-generator

# install
cd React-generator && npm install

# dev for client
npm run dev

# dev for ssr
npm run devs
```
